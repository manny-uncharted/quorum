The insight

Predict's price is a probability — but a driftless one.

DeepBook Predict mints binary options against an oracle's volatility surface. That surface is risk-neutral: it assumes zero drift. So re-pricing it gives you no edge. The edge is in estimating the real-world probability it ignores — momentum, order flow, funding, catalysts. And today's on-chain trading bots are black boxes. Quorum fixes both.

Transparent by construction

Every decision streams live — analyst signals, bull/bear debate, the trader's probability, the edge, the risk verdict — and is sealed into a signed, content-hashed evidence bundle. Auditable, not a black box.

Disciplined by design

A deterministic quant core owns the arithmetic: edge = Psubjective − Pmarket, fractional Kelly sizing, and a code-owned risk gate (time-to-expiry, vol regime, exposure, daily-loss kill switch). It abstains when there's no edge — which is most of the time.

The primitive

Quorum doesn't just consume Predict — it emits a new on-chain feed.

Every run publishes its consensus to a shared ConsensusOracle object in our own quorum_oracle Move package. Sui has no forward-looking, reasoning-derived probability benchmark today. This is one — readable by any protocol, in basis points, backed by the evidence hash that produced it.

ConsensusOracle · quorum_oracle::consensus

Cap-gated publish, keyless read, a ConsensusPublished event for indexers. Publishing needs only the desk key — no DUSDC — so the feed works even in the keyless demo.

How it works

One pipeline, fully event-sourced.

The same loop runs for three personas — a keyless beginner (heuristic, paper), an analyst (their own probability), and a trader (full Gemini debate). Every stage emits a typed event the CLI, the live UI, and the evidence bundle all consume.

01

Analysts

Volatility · momentum · catalyst · flow reason over the live SVI surface + market data.

02

Debate

A bull and bear stress-test the view.

03

Trader

Synthesizes one calibrated P(up).

04

Quant

Edge vs. market, fractional Kelly sizing — code-owned, unit-tested.

05

Risk gate

Vetoes / resizes on regime, exposure, expiry, daily-loss.

06

Execute

Real mint on DeepBook Predict testnet via a PredictManager.

07

Publish ▸ on-chain

Consensus written to the ConsensusOracle as a reusable primitive.

Composability

A feed other protocols build on.

Option vaults to price and skew, liquidation engines, other desks — any of them can read Quorum's consensus. We ship a working downstream consumer: an option vault that sizes a directional tilt from the feed, discounted by the disagreement index.
